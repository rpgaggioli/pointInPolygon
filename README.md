Point-In-Polygon
====================

## Install


## API

### `require(" ")( P , q)`

Checks if point q is contained within the polygon P. 

* `P` is an array of ordered vertices that define the polygon. 
* `q` is a point.

**Returns** 1 if the q is in P. 0 if q is not in P.
